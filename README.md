Build Instructions:
-------------------

Pip install (into a virtual environment):

    pip install catalogue-flask

or
    
    pip install git+https://github.com/ScottWales/catalogue-flask

or (for development)

    git checkout git+https://github.com/ScottWales/catalogue-flask
    cd catalogue-flask
    pip install -e '.[dev]'
